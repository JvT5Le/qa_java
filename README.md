# Yandex Practicum AQA Java course
## 6 Sprint final project
At this project we will use the following tools:

Java 11;
Maven 3.8.1;
Junit 4.13.2;
Mockito 5.3.1
Jacoco 0.8.10

Run the project:
```shell
mvn clean test
```
Make a test coverage jacoco report:
```shell
mvn verify
```
